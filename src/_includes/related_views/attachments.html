Add Attachment

Use this action to create a new attachment for the selected work order.

  1. Select the New Record icon (plus symbol) on the action bar. {{ site.data.ds.product-name }} opens the Attachments dialog.
  2. Set a Description value. The description should be short but informative.
  3. Set the Document Folder value.
  4. Add a file in one of the following ways:
    • Online: Set the URL value to point to a network accessible online resource and set the URL Type to URL.
    • Upload file: Tap or click the the Upload File icon in the Document Data field and select a file to upload from local storage using the document browser.
    • Take photo: If you are using a laptop or mobile device, you can tap or click the Take Photo icon to access the device's camera to take a picture.
  5. Tap or click Done (check mark on smaller screens). Alternatively, tap or click Cancel (x icon on smaller screens) to close the dialog without saving the attachment.
  6. {{ site.data.ds.product-name }} returns to the Attachments related view. If you uploaded a file or took a photo, the Document Data field now has a preview icon. Tap or click the icon to view the file. If you created an online reference, the URL field becomes an active link that navigates a new browser tab to the referenced URL.
